---------------------------------------------
Step 1: Create namespace
kubectl apply -f k8s/namespace.yaml
Step 2: Deploy database first
kubectl apply -f k8s/postgres/
Check:
kubectl get pods -n fraud-detection
kubectl get svc -n fraud-detection
Step 3: Deploy MinIO
kubectl apply -f k8s/minio/
Step 4: Deploy MLflow
kubectl apply -f k8s/mlflow/
MLflow needs PostgreSQL + MinIO ready first.
Step 5: Deploy FastAPI
kubectl apply -f k8s/fastapi/
FastAPI needs MLflow + PostgreSQL.
Step 6: Deploy Dashboard
kubectl apply -f k8s/dashboard/
Dashboard needs FastAPI/PostgreSQL.
Step 7: Deploy Airflow
kubectl apply -f k8s/airflow/
Airflow needs PostgreSQL and access to your ML code.
Step 8: Deploy monitoring
kubectl apply -f k8s/prometheus/
kubectl apply -f k8s/grafana/

7. Simple rule
Use this rule:
One application/image = one Deployment + one Service
Stateful app = StatefulSet + PVC + Service
Browser access = Ingress
Sensitive values = Secret
Normal config = ConfigMap
For your project:
PostgreSQL в†’ StatefulSet
MinIO      в†’ StatefulSet
MLflow     в†’ Deployment
FastAPI    в†’ Deployment
Dashboard  в†’ Deployment
Airflow    в†’ multiple Deployments
Prometheus в†’ Deployment or Helm
Grafana    в†’ Deployment + PVC
