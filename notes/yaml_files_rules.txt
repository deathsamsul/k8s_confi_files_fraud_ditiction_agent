Quick summary
kubectl get svc → services status
kubectl get pods → app health
kubectl get deployments → rollout status
kubectl get all → everything in one shot
systemctl status kubelet → VM-level service



delete cluster
kubectl delete all --all --all-namespaces
Check if cluster is running
kubectl cluster-info


Check namespaces
kubectl get namespaces
or short:

kubectl get ns
Check all services in all namespaces
kubectl get svc -A

Check all pods in all namespaces
kubectl get pods -A
